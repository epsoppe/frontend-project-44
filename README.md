## Описание

**Brain Games** - это набор из 5 консольных игр для развития математического мышления и логики:

1. **Проверка на чётность** - определите, является ли число чётным
2. **Калькулятор** - вычислите результат арифметического выражения
3. **НОД (Наибольший общий делитель)** - найдите НОД двух чисел
4. **Арифметическая прогрессия** - найдите пропущенное число в прогрессии
5. **Простое число** - определите, является ли число простым

## Минимальные требования

- Node.js версии 13.2.0 или выше
- npm версии 6.13.0 или выше
